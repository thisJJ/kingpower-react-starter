# สร้าง React App


ทำความเข้าใจการติดตั้ง และโครงสร้างของ React


## ติดตั้ง
```
npx create-react-app my-app
cd my-app
npm start
```

## โครงสร้างไฟล์เริ่มต้น

```
my-app
├── README.md
├── node_modules
├── package.json
├── public
│   ├── favicon.ico
│   ├── index.html
│   └── manifest.json
└── src
    ├── App.css
    ├── App.js
    ├── App.test.js
    ├── index.css
    ├── index.js
    ├── logo.svg
    └── registerServiceWorker.js
```

## เริ่มใช้งาน

```
cd my-app
```

```
npm run start
```

เปิด http://localhost:3000/
