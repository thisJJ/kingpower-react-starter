#### การทำงาน
1. ผู้ใช้งานสามารถป้อนข้อมูลลงใน Text input ได้
2. กด + เพื่อเพิ่มข้อมูลไปยัง redux store
3. ข้อมูลที่ทำการเพิ่ม จะแสดงในรายการทันที
4. ผู้ใช้งานกด X เพื่อลบข้อมูลที่เลือก
